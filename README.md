# Sales Data Analysis using Excel

## Project Overview
This project focuses on analyzing sales data using Microsoft Excel.  
The analysis was performed using data cleaning techniques, Pivot Tables, Pivot Charts, and an interactive dashboard to identify important business insights.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Dashboard
- Data Cleaning Techniques

---

## Project Workflow
1. Data Cleaning
2. Handling Missing Values
3. Converting Text to Lowercase
4. Creating Pivot Tables
5. Data Visualization using Charts
6. Dashboard Creation
7. Business Insights Generation

---

## Key Insights

- North region recorded the highest sales count.
- Central region generated the highest sales revenue.
- Punjab had the highest sales count among all states.
- Tamil Nadu generated the highest sales amount.
- Beauty and Clothing were the most preferred categories.
- Home Office segment contributed the highest sales.
- UPI was the most preferred payment method.
- Same Day was the most used shipping mode.
- Most customers provided ratings of 4 and 5, indicating high satisfaction.
- Around 83% orders were not returned.
